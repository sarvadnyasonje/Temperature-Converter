# Temperature Converter

## Description

A C++ console-based Temperature Converter that converts temperature values between Celsius, Fahrenheit, and Kelvin.

## Features

* Celsius to Fahrenheit
* Celsius to Kelvin
* Fahrenheit to Celsius
* Fahrenheit to Kelvin
* Kelvin to Celsius
* Kelvin to Fahrenheit
* User-friendly menu
* Exit option
* Function-based implementation

## Technologies Used

* C++
* VS Code

## How to Run

1. Compile the code:
   g++ main.cpp -o converter

2. Run the program:
   ./converter
